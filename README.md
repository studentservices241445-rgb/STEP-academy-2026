## ✅ الميزات المُنفَّذة

### 🎨 التصميم
- تصميم RTL كامل باللغة العربية
- ألوان متدرجة (ذهبي + أزرق داكن + أبيض)
- خطوط Google Fonts (Cairo + Tajawal)
- تأثيرات جسيمات متحركة في Hero Section
- شريط إشعارات متحرك علوي
- عداد تنازلي للعرض المحدود
- تصميم متجاوب كامل (Mobile Responsive)

### 🛒 سلة التسوق
- إضافة الدورات من بطاقات المنتجات أو جدول المقارنة
- Sidebar للسلة مع عرض المنتجات
- حساب تلقائي للمجموع والتوفير
- حفظ السلة في LocalStorage
- حذف الدورات من السلة

### 📦 عملية الشراء (3 خطوات)
1. **بيانات العميل**: الاسم + الهاتف + البريد + الدرجة المستهدفة
2. **التحويل البنكي**: عرض بيانات بنك الإنماء + نسخ بنقرة
3. **إرفاق الإيصال**: رفع صورة + إرسال لتيليجرام

### 📱 إرسال الطلب
- رابط تيليجرام جاهز مع رسالة منسقة كاملة
- البيانات: اسم + هاتف + بريد + درجة + الدورات + المجموع + الإيصال
